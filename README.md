## 프로젝트 소개
대법원 및 온비드의 경·공매 데이터를 수집·가공하여 투자 정보를 제공하는 데이터 파이프라인 프로젝트입니다.
비정형 데이터를 크롤링하여 정형 데이터로 가공하고, 이를 DB에 적재하는 ETL 파이프라인을 구축했습니다.  
또한 Apache Airflow를 활용하여 데이터 수집 및 처리 과정을 자동화하고, Scikit-learn 기반 회귀 모델을 통해 예상 낙찰가를 산출하도록 설계했습니다.

## 아키텍처

                ↓

## 기술스택
- Python
- Apache Airflow
- Docker
- PostgreSQL
- BeautifulSoup
- Scikit-learn

## 데이터 수집
- 온비드 및 대법원 경·공매 데이터 크롤링 및 각종 데이터 크롤링(물건정보,현황조사서,파일데이터 등 주요데이터수집)
- 페이지네이션 기반 대량 데이터 수집
- HTTP 요청 실패 대비 retry 및 timeout 처리

## 데이터 파이프라인 (Airflow)
- DAG 기반 ETL 자동화
- 하루 2회 스케줄링
- Branching(분기 처리)를 통한 데이터 유무 기반 흐름 제어
- 수집된 데이터를 기존 DB와 비교하여 신규 및 변경 데이터 식별
- 신규 데이터는 상세 크롤링 수행 후 DB 적재
- 변경 데이터는 최신 정보를 반영하기 위해 재크롤링 후 업데이트 처리
- retry / timeout 설정을 통한 안정성 확보
- 로그 자동 정리 자동화
